Problem 1 :¶

By performing matrix multiplication between an ((n, k)) matrix (A) and a ((k, m)) matrix (B), let the resulting ((n, m)) matrix be (C). The elements (c_{ij}) in matrix (C) are given by:

$$ c_{ij} = \sum_{l=1}^{k} a_{il} \cdot b_{lj} $$

No. of Multiplications¶

For each element (c_{ij}) in matrix (C):

  • Number of Multiplications: we need to compute (k) products.

Since there are (n * m) elements in the resultant matrix (C):

  • Total Multiplications: Each element requires (k) multiplications.

    $$ \text{Total No of Multiplications} = n \times m \times k $$

No. of Additions¶

For each element (c_{ij}) in matrix (C):

  • Number of Additions: Adding (k) terms requires (k - 1) additions.

Since there are (n * m ) elements in matrix (C):

  • Total Additions: Each element requires (k - 1) additions.
$$ \text{Total Additions} = n \times m \times (k - 1) $$

import time

def matrix_multiplication(A, B):
    n = len(A)
    k = len(A[0])
    m = len(B[0])


    C = [[0] * m for _ in range(n)]

    for i in range(n):
        for j in range(m):
            sum = 0
            for l in range(k):
                sum += A[i][l] * B[l][j]
            C[i][j] = sum

    return C

Problem 4 :¶

Given function: $$( f(x, y) = x^2 y + y^3 \sin(x) ).$$

The partial derivative of ( f(x, y) ) with respect to ( x ) is:

$$ \frac{\partial f}{\partial x} = 2xy + y^3 \cos(x) $$

The partial derivative of ( f(x, y) ) with respect to ( y ) is:

$$ \frac{\partial f}{\partial y} = x^2 + 3y^2 \sin(x) $$

The gradient of ( f(x, y) ) is:

$$ \nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right) = \left( 2xy + y^3 \cos(x), x^2 + 3y^2 \sin(x) \right) $$

def defined_argmin(array):
    array==np.asarray(array)
    i=0
    j=1

    while j<=len(array)-1:
        if array[j]<=array[i]:
            i=j
            j+=1
        j+=1
    return i
